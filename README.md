# The Unfair Climate Price

> A visual data story about how Pacific island nations are paying the highest price for a climate crisis they barely caused. Built for the Pacific Dataviz Challenge 2026.[cite: 6]

## Overview

Small Pacific island nations barely contribute to climate change, producing less than 1% of global emissions.[cite: 6] However, six years of official data show they are losing the most land, money, and coastline because of it.[cite: 6] "The Unfair Climate Price" is an interactive data journalism project that visualizes this structural climate inequality across the Pacific.[cite: 6]

**If you are interested in data storytelling, front-end accessibility in dataviz, or climate communication, please consider starring this repository.**

## Key Features for Data Practitioners

This project serves as a practical case study for building accessible, high-performance web visualizations:
* **Scroll-Driven Narrative:** Uses the browser's native IntersectionObserver to animate sections into view seamlessly as the user scrolls.[cite: 6]
* **Web Accessibility:** Every interactive component works with a keyboard and is optimized for screen readers.[cite: 6]
* **Motion Sensitivity:** The entire site respects user-defined reduced-motion system settings.[cite: 6]
* **Scalable Vectors:** Custom charts are built to be drawn as scalable graphics that stay sharp at any screen size.[cite: 6]

## Data Sources

The narrative is driven by six open datasets sourced directly from the Pacific Data Hub's .Stat Explorer:[cite: 6]
1. Sea level anomalies[cite: 6]
2. Mean sea surface temperature anomalies[cite: 6]
3. Greenhouse gas emissions per capita[cite: 6]
4. Direct disaster economic loss[cite: 6]
5. Coastline[cite: 6]
6. Climate altering land cover index[cite: 6]
