Politique de Confidentialité

Comment les données sont gérées sur ce projet interactif.

Dernière mise à jour : 3 juillet 2026

1. Pas de Suivi ni de Cookies

The Pacific Paradox est conçu comme un récit de données statique pour le Pacific Dataviz Challenge 2026. Nous respectons votre vie privée dès la conception. Nous n'utilisons pas de scripts d'analyse, de pixels publicitaires ou de traceurs tiers. Nous ne stockons pas de cookies sur votre appareil.

2. Journaux du Serveur

Ce projet est hébergé sur une plateforme de déploiement web standard. Le fournisseur d'hébergement collecte automatiquement des journaux de serveur de base et anonymisés (qui peuvent inclure votre adresse IP, votre type de navigateur et l'horodatage) strictement à des fins de sécurité et pour garantir que le site reste en ligne. Ces données ne sont pas utilisées pour le profilage ou le marketing.

3. Liens Externes

Ce site contient des liens vers le Pacific Data Hub et d'autres ressources externes à des fins de citation. Une fois que vous cliquez sur un lien et quittez ce site, vous êtes régi par les politiques de confidentialité et les pratiques de collecte de données de ces plateformes spécifiques.

4. Open Source

Le code de ce projet est open source. Toutes les interactions que vous avez avec le dépôt sous-jacent (par exemple, GitHub) sont régies par la politique de confidentialité de cette plateforme.

À propos du projet

Ce récit de données a été créé pour le Pacific Dataviz Challenge 2026, sur le thème du changement climatique. Les visualisations sont conçues pour mettre en évidence les inégalités climatiques structurelles à travers le Pacifique.

Les sources de chaque graphique se trouvent en dessous de celui-ci. Tous les jeux de données référencés sont des données ouvertes. Le code source est disponible sur GitHub.

Sources de données

  • Anomalies du niveau de la mer & de la température de surface Pacific Data Hub
  • Émissions de GES par habitant Pacific Data Hub
  • Pertes économiques & trait de côte / couverture terrestre Pacific Data Hub

Technologies et outils

D3.js v7, p5.js, RawGraphs, Vanilla JS, Adobe Illustrator, Adobe Photoshop, SDMX / .Stat
