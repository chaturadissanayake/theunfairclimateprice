Privacy Policy

How data is handled on this interactive project.

Last updated: July 3, 2026

1. No Tracking or Cookies

The Unfair Climate Price is built as a static data story for the Pacific Dataviz Challenge 2026. We respect your privacy by design. We do not use analytics scripts, advertising pixels, or third-party trackers. We do not store cookies on your device.

2. Server Logs

This project is hosted on a standard web deployment platform. The hosting provider automatically collects basic, anonymized server logs (which may include your IP address, browser type, and timestamp) strictly for security purposes and to ensure the site remains online. This data is not used for profiling or marketing.

3. External Links

This site contains links to the Pacific Data Hub and other external resources for citation purposes. Once you click a link and leave this site, you are governed by the privacy policies and data collection practices of those specific platforms.

4. Open Source

The code for this project is open-source. Any interactions you have with the underlying repository (e.g., GitHub) are governed by that platform's respective privacy policy.

About the project

This data story was created for the Pacific Dataviz Challenge 2026, themed around climate change. The visualizations are built to surface structural climate inequality across the Pacific.

Sources for each chart sit beneath it. All datasets referenced are open data. Source code is available on GitHub.

Data sources

  • Sea level & sea surface temperature anomalies Pacific Data Hub
  • GHG emissions per capita Pacific Data Hub
  • Disaster economic loss & coastline / land cover index Pacific Data Hub

Tech & tools used

D3.js v7, p5.js, RawGraphs, Vanilla JS, Adobe Illustrator, Adobe Photoshop, SDMX / .Stat
